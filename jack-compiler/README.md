# Jack Compiler

Written in C++.

Lexical analysis: all tokens are either *keywords*, *symbols*, *integer literals*, *string literals* or *identifiers*.

Grammar: 
- Defines the expected form for constructs like let statements, if-statements, while loops, etc.
